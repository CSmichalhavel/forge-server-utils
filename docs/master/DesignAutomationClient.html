Class: DesignAutomationClient

DesignAutomationClient(auth, hostopt, regionopt)

Client providing access to Autodesk Forge design automation APIs.

Constructor

new DesignAutomationClient(auth, hostopt, regionopt)

Initializes new client with specific authentication method.

Parameters:
Name Type Attributes Default Description
auth IAuthOptions

Authentication object, containing either client_id and client_secret properties (for 2-legged authentication), or a single token property (for 2-legged or 3-legged authentication with pre-generated access token).
host string <optional>
 "https://developer.api.autodesk.com"

Forge API host.
region Region <optional>
 "US"

Forge availability region.
