BIM360Client

Client providing access to Autodesk Forge BIM360 APIs.

Constructor

Initializes new client with specific authentication method.

Parameters:
Name Type Attributes Default Description
auth object <optional>
 {token: FORGE_ACCESS_TOKEN}

Authentication object, containing either client_id and client_secret properties (for 2-legged authentication), or a single token property (for authentication with pre-generated access token).
host string <optional>
 "https://developer.api.autodesk.com"

Forge API host.
