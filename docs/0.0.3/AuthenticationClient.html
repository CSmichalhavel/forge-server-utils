AuthenticationClient

AuthenticationClient

Client providing access to Autodesk Forge authentication APIs. https://forge.autodesk.com/en/docs/oauth/v2

Constructor

new AuthenticationClient(client_idopt, client_secretopt)

Initializes new client with specific Forge app credentials.

Parameters:
Name Type Attributes Description
client_id string <optional>

Forge application client ID. If not provided, the constructor will attempt to get the value from env. variable FORGE_CLIENT_ID.
client_secret string <optional>

Forge application client secret. If not provided, the constructor will attempt to get the value from env. variable FORGE_CLIENT_SECRET.

Methods

authenticate(scopes, forceopt) → {Promise.<string>}

Retrieves 2-legged access token for a specific set of scopes. Unless the {@see force} parameter is used, the access tokens are cached based on their scopes and the 'expires_in' field in the response.

Parameters:
Name Type Attributes Default Description
scopes Array.<string>

List of requested scopes.
force boolean <optional>
 false

Skip cache, if there is any, and retrieve a new token.
